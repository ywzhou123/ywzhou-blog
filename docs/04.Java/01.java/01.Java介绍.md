# Java 介绍

## 概念

Java SE：面向对象、api、jvm

Java EE：jsp、ejb、服务

Java ME：移动设备、通信、游戏



Jvm：java虚拟机，将.java编译成.class，通过解释器翻译成计算机显示的内容

Jdk：java开发工具包

Jre：java运行时环境



## 安装

安装jdk

配置环境变量

- JAVA_HOME 安装路径

- PATH 命令路径 

- CLASSPATH 类库路径
