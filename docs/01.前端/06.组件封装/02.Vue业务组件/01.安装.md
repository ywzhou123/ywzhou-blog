# 开发指南

## 安装

​		需要安装`th-vue-component`

```shell
npm i element-ui -S
npm i xlsx -S
npm i th-vue-component -S
npm i th-vue-business -S
```


## 更新

```shell
npm update th-vue-business -S
```



## 引入

在 main.js 中写入以下内容：

```javascript
import Vue from 'vue'
import App from './App'
import ThBusiness from 'th-vue-business'
import 'th-vue-business/lib/th-vue-business.css'

Vue.use(ThBusiness)

new Vue({
  render: h => h(App)
}).$mount('#app')
```
